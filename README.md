# DLMC-SPARC Analysis & MORPHEUS Framework

This repository contains computational notebooks and scripts for both:  
- **DLMC (Dynamic Local Mass Clustering) analysis** of SPARC galaxy rotation curves  
- **MORPHEUS reaction-diffusion framework** for biological pattern formation  

It provides a unified platform bridging **astrophysical field dynamics** and **biological reaction-diffusion systems**.

---

## DLMC Framework (Galactic Flux Analysis)

Using the **FluxCore v5 framework**, DLMC models unified dark matter dynamics and validates rotational velocity profiles against SPARC data.  

The framework integrates **baryonic contributions, radial acceleration relations (RAR), and gravitational lensing effects**, providing a reproducible and modular tool for astrophysical research in modified gravity and galactic dynamics.
